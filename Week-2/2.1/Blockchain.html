BLOCKCHAIN

It is a real-time peer to peer distributed immutable database.

Blockchain Uses

Blockchain has many Uses

  1. Open System: To create an open system with no more solied or centralized and closed. This data is not controlled by intermediaries.
  2. Trustless: When there are no intermediaries and rules are set by sytems and computers that cannot be broken.
  3. Represent Value: The data is distributed and served. Everyone gets a copy of data. So everyone can see if the value changes.

We use blockchain to solve a problem. Blockchain solves problem of usage of closed, siloed or centralized systems, trusted intermediaries, representing values. To represent money with traditional system. We need to database and every order placed will be recorded and maintained by someone. It is very transparent. It can store any kind of value.

Ethereum

It is a Blockchain application platform. It allows to run programs through smart contracts.

How does Ethereum works?
Its a Blockchain with native crypto currency, that can be sent, bought, in some exchanges mined and stored. It can be habour wallets and cold storage. It can run smart contracts.

How to use Ethereum?
Inorder to use Ethereum we need an account which can be capable of track balances that how many Ethereum coins are there in the wallet.
Account is comprised of set of key pairs.

  1. Public Key: This is like password for network.
  2. Private Key: This is like Username and also called as Ethereum address.
